# IPM-wolf-altominho-portugal
Data and R Code for the manuscript entitled Assessing key demographic parameters and viability of a wolf population in a highly human-dominated landscape, published in 'Biological Conservation'

The processed data file ("data_IPM_wolf_AM_PT.RData") includes the data frame with annual population size (C), annual number of female breeders (y_female), and the
capture history with states (rCH) of the 205 individuals genetically identified in the study area of Alto Minho (Portugal), between 2007 and 2019.


The R Code ("IPM_Nakamura_etal._2026.Rmd") includes data input and the integrated population model to run in JAGS, using the R package 'R2jags'.
